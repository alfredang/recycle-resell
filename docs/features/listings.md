# Listings

Create and manage marketplace listings.

## Creating a Listing

1. Click "Start selling" or navigate to `/listing/create`
2. Sign in if not already authenticated
3. Fill in the listing details

### Required Fields

| Field | Description |
|-------|-------------|
| **Title** | 3-100 characters |
| **Description** | 10-5000 characters |
| **Price** | Amount in SGD |
| **Category** | Select from available categories |
| **Condition** | Item condition |
| **Location** | Meetup location |
| **Images** | At least 1 image (max 10) |

### Condition Options

| Condition | Description |
|-----------|-------------|
| New | Brand new, unused |
| Like New | Excellent condition, barely used |
| Good | Good condition, minor wear |
| Fair | Fair condition, visible wear |
| Poor | Poor condition, significant wear |

### Optional Fields

- **Brand** - Item brand name
- **Tags** - Keywords for search
- **Delivery** - Can deliver / meetup only
- **Delivery Fee** - If delivery is available

## Uploading Images

- Drag and drop or click to upload
- Supported formats: JPG, PNG, WebP
- Maximum 10 images per listing
- First image is the cover photo
- Reorder by dragging

## Listing Status

| Status | Description |
|--------|-------------|
| Available | Active and visible |
| Reserved | Reserved for a buyer |
| Sold | Transaction completed |
| Draft | Not published |

## Managing Listings

### View Your Listings

- Navigate to `/listings` or user profile
- See all your listings with status

### Edit a Listing

1. Click on your listing
2. Click "Edit"
3. Make changes
4. Save

### Delete a Listing

1. Click on your listing
2. Click "Delete"
3. Confirm deletion

### Mark as Sold

1. Click on your listing
2. Click "Mark as Sold"
3. Select the buyer (if from chat)

## Listing Detail Page

The listing detail page shows:

- Image gallery with carousel
- Title, price, condition
- Description
- Seller information
- Location
- Action buttons (Contact, Favorite)

## Best Practices

### Good Titles

- Be specific: "iPhone 14 Pro 256GB Space Black"
- Include key details
- Avoid ALL CAPS

### Good Descriptions

- Describe condition honestly
- List any defects
- Include dimensions if relevant
- Mention original accessories

### Good Photos

- Use good lighting
- Show all angles
- Include close-ups of defects
- Use clean backgrounds
